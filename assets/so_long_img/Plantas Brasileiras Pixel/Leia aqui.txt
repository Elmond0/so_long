Oie, meu nome é Maria Giovana e este é meu projeto de Iniciação
Artistica pela UFJF.
Trago neste projeto sprites gratuitas de 50 plantas nativas brasileiras.
Todas as plantas (salvo algumas excessões) contam com uma versão de muda
(planta pequena), uma versão da planta adulta e um item que esta planta
pode gerar (fruto, flor, folha). Todos os desenhos foram feitos na proporção
32x32 pixels.

A pasta "plantas" contém os desenhos finalizados, já o pdf "referência 
plantas" contém uma tabela feita para o projeto com o nome popular das
plantas, nome ciêntifico, forma de vida e o site que foi usado como referência.
Este é meu primeiro projeto trabalhando com conteúdo para jogos, então 
estou aberta a feedbacks!
Espero que gostem!
